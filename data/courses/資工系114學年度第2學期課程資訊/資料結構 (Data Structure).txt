第12週課程 (2026/05/10─2026/05/16)：Ch8-2 Insert, delete, and process nodes in an AVL tree Ch8-3 Understand the operation of the AVL tree ADT Ch8-4 Write application programs using an AVL tree
第13週課程 (2026/05/17─2026/05/23)：Ch9-1 Define and implement heap structures Ch9-2 Understand the operation and use of the heap ADT Ch9-3 Design and implement selection applications using a heap Ch9-4 Design and implement priority queues using a heap 
第14週課程 (2026/05/24─2026/05/30)：Ch10-1 Define and discuss multiway tree structures Ch10-2 Understand the operation and use of the B-tree ADT Ch10-3 Discuss the use and implementation of simplified B-trees Ch10-4 Compare and contrast B-trees, B*trees, and T+trees
