課程名稱：硬體描述語言程式設計與模擬 (System Design Using Hardware Description Language)
部別：日間部研究所碩士班
開課班級：資工碩一
學分數：3.0學分
授課時數：3.0小時
授課教師：王建鈞
必選修：選修
教室：E221崁入式系統與機器
修課上限人數：30人
上課時間：每周星期三上午第2節~第4節

課程教學目標：
1. 理解硬體描述語言的基本概念和歷史。
2. 熟悉VHDL的語法 and 結構。
3. 掌握數位電路設計流程，包括編碼、合成、模擬和驗證。
4. 能夠使用硬體描述語言設計基本數位電路。
5. 能夠進行電路模擬，分析和驗證電路行為。
6. 掌握設計實踐項目的能力，能夠解決實際設計問題。
7. 培養細心、嚴謹的設計態度。
8. 鼓勵創新思維，嘗試不同的設計方案。
9. 增強團隊合作和交流能力。

課程教學綱要：
1. 本課程旨在介紹硬體描述語言（HDL）如Verilog或VHDL，並教授如何使用這些語言進行數位電路的設計和模擬。課程內容包括基本語法、設計流程、模擬技術以及實踐項目，幫助學生掌握設計和驗證數位電路的核心技能。
