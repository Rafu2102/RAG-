教學進度表（2023年秋季學期）：
第1週課程 (2023/09/10─2023/09/16)：Introduction to Data structure
第2週課程 (2023/09/17─2023/09/23)：Ch2-(1)Explain the difference between iteration and recursion ch2-(2)Design a recursive algorithm Determine when an recursion is an appropriate solution ch2-(3)Write simple recursive functions
第3週課程 (2023/09/24─2023/09/30)：Ch3-(1)Explain the design, use, and operation of a stack Ch3-(2)Implement a stack using a linked list structure Ch3-(3) Understand the operation of the stack ADT Ch3-(4) Write application programs using the stack ADT
第4週課程 (2023/10/01─2023/10/07)：Ch3-(5) Discuss reversing data, parsing, postponing and backtracking Ch4-(1)Explain the design, use, and operation of a queue Ch4-(2)Implement a queue using a linked list structure Ch4-(3)Understand the operation of the queue ADT
第5週課程 (2023/10/08─2023/10/14)：Ch4-(4) Write application programs using the queue ADT Ch4-(5)Explain categorizing data and queue simulation
第6週課程 (2023/10/15─2023/10/21)：Ch5-1 Explain the design, use, and operation of a linear list Ch5-2 Implement a linear list using a linked list structure
第7週課程 (2023/10/22─2023/10/28)：Ch5-3 Understand the operation of the linear list ADT Ch5-4 Write application programs using the linear list ADT Ch5-5 Design and implement different link-list structures
第8週課程 (2023/10/29─2023/11/04)：Middle Exam
第9週課程 (2023/11/05─2023/11/11)：Ch6-1 Understand and use basic tree terminology and concepts Ch6-2 Recognize and define the basic attributes of a binary tree Ch6-3 Process trees using depth-first and breadth-first traversals Ch6-4 Parse expressions using a binary tree
第10週課程 (2023/11/12─2023/11/18)：Ch6-5 Design and implement Huffman trees Ch6-6 Understand the basic use and processing of general trees CH7-1 Create and implement binary search trees CH7-2 Understand the operation of the binary search tree ADT
第11週課程 (2023/11/19─2023/11/25)：CH7-3 Write application programs using the binary search tree ADT CH7-4 Design and implement a list using a BST CH7-5 Design and implement threaded trees Ch8-1 Explain the differences between a BST and an AVL tree
第12週課程 (2023/11/26─2023/12/02)：Ch8-2 Insert, delete, and process nodes in an AVL tree Ch8-3 Understand the operation of the AVL tree ADT Ch8-4 Write application programs using an AVL tree
第13週課程 (2023/12/03─2023/12/09)：Ch9-1 Define and implement heap structures Ch9-2 Understand the operation and use of the heap ADT Ch9-3 Design and implement selection applications using a heap Ch9-4 Design and implement priority queues using a heap
第14週課程 (2023/12/10─2023/12/16)：Ch10-1 Define and discuss multiway tree structures Ch10-2 Understand the operation and use of the B-tree ADT Ch10-3 Discuss the use and implementation of simplified B-trees Ch10-4 Compare and contrast B-trees, B*trees, and T+trees
第15週課程 (2023/12/17─2023/12/23)：Ch10-5 Discuss the design and use a lexical search trees (Tries) Ch11-1 Understand and use basic graph terminology and concepts Ch11-2 Define and discuss basic graph and network structures Ch11-3 Design and implement graph and network applications
第16週課程 (2023/12/24─2023/12/30)：Ch11-4 Design and implement applications using the graph ADT Ch11-5 Define and discuss Dijkstra's shortest path algorithm Ch12-1 Understand the basic concepts of internal and external sorts Ch12-2 Discuss the relative efficiency of different sorts
第17週課程 (2023/12/31─2024/01/06)：Ch12-3 Recognize and discuss selection, insertion and exchange sorts Ch12-4 Discuss the design and operation of external sorts Ch12-5 Recognize natural, balanced, and polyphase merge sorts
第18週課程 (2024/01/07─2024/01/13)：Final Exam
