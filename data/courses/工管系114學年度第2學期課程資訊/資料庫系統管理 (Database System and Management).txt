課程教學目標：
1. 本課程旨在培養工管系學生具備建構企業級資料庫系統的核心職能。在智慧製造與數位轉型的浪潮下，高效的數據管理是工業工程專業不可或缺的一環。課程重點分為三大面向：首先是設計邏輯訓練，掌握實體關聯模型（ER Model）繪製與正規化理論，確保資料架構的正確性。其次是開發技術實務，精通 SQL 語法、預存程序及觸發程序，並學習如何運用 ChatGPT 輔助編寫複雜查詢，提升開發效率。最後是系統整合應用，引導學生結合 Python 與 PHP 建立資料庫用戶端介面，使學生具備從後端設計到前端應用的全方位數據管理能力，協助企業落實數據驅動決策。

課程教學綱要：
1. 本課程結合資料庫理論與產業實務，透過實作引導學生掌握 MySQL/MariaDB 的應用開發。內容涵蓋：理論與設計基礎（介紹關聯式資料庫模型、ER 模型設計工具及正規化程序，降低資料冗餘）、資料庫建置實務（學習 SQL 語言、建立資料表及設定完整性限制條件）、進階開發與優化（深入探討檢視表、索引規劃及交易處理，確保系統效能與安全）以及 AI 輔助與系統整合（整合 ChatGPT 進行 SQL 指令優化，並利用 Python 與 PHP 進行用戶端開發實作）。
