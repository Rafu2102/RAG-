課程教學目標：
1. 本課程為 Python 程式設計入門，適合資訊技術初學者，旨在培養學生對程式語言的基本理解，提升邏輯思考與問題解決能力。課程涵蓋 Python 基礎語法、擴充應用與實作練習。最後期望學生能獨立撰寫程式並完成指定作業。

課程教學綱要：
1. 本課程將帶領學生進入 Python 程式設計的基礎世界，從 Python 環境安裝與系統介面操作開始，逐步建立程式設計能力。課程內容涵蓋變數與數學運算、基本輸入與輸出、流程控制（if 條件判斷、迴圈設計）、函式開發與介面設計等主題。同時，將介紹運算思維（Computational Thinking）與程式演算法流程，幫助學生培養基礎程式邏輯概念，奠定未來學習進階課程的基石，如資料科學分析、人工智慧與機器學習演算法等領域。
